Winkler Puzzle 2: Summing Sets

2025/02/06

Here’s another puzzle from Mathematical Puzzles: a Connoiseur’s Collection:

Prove that in a set of 25 numbers, it is possible to select two numbers such that no other number in the set is equal to the sum or difference of the two you selected

Warning: spoilers ahead! There’s too much here to wrap it all in a spoiler block.

Winkler’s solution is:

Let the numbers be $x_1 < x_2 < \cdots < x_n$. If $x_n$ is unavailable to be taken as one of the desired numbers, it must be that for each lower number $x_i$, there is another $x_j$ with $x_i + x_j = x_n$. Thus, the first 24 numbers are paired in such a way that $x_i + x_{n-i} = x_n$. Now consider $x_{n-1}$ together with any of $x_2, \ldots, x_{n-2}$; these pairs sum to more than $x_n = x_{n-1} + x_1$ and so $x_2, \ldots, x_{n-2}$ must also be paired, this time satisfying $x_{2+i} + x_{n-2-i} = x_{n-1}$. But that leaves $x_{(n-1)/2}$ paired with itself, so the numbers $x_n-1, x_{(n-1)/2}$ solve the problem.

I think this doesn’t quite work! The problem is that in the second step, $x_2, \ldots, x_{n-2}$ don’t actually have to be paired with each other, they have to be paired with any number in $x_1, \ldots, x_{n-2}$. So another solution is needed.
